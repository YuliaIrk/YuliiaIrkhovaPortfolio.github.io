Covid data exploration

In this project I was exploring data on Covid from open resources. This project was also done with MySQL. Project Funnel Onboarding

In this project, the onboarding stages of the user starting from registration to the purchase of the product were investigated using Tableau. Also with the help of calculated fields and quick table calculations, I counted conversion rates from registration to trial version and also “Average days from registration to” each of the stages of onboarding. With the help of Tableau I created an interactive dashboard with a funnel chart, which served as a filter for other visualizations.
